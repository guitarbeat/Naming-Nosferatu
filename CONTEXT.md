# Naming Nosferatu

An interactive web application where users run head-to-head tournament matchups to rank their favorite cat names and view detailed analytics.

## Language

**Shortlist**:
A curated subset of cat names chosen by a user from the active name pool to enter the tournament bracket.
_Avoid_: Selected names, picks, selection, top picks

**Rating**:
The Elo-based quantitative score of a name item calculated from head-to-head tournament matchups.
_Avoid_: Elo, score, points, stats
